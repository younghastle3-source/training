

# **前鋸筋（ぜんきょきん）の役割と重要性**

## **前鋸筋の基本的な役割と解剖学的特徴**

* 前鋸筋は、トレーニングにおけるプッシュやプルの動作、スポーツ現場、姿勢改善など、上肢の筋肉の中でも特に活躍する場面が多い筋肉です 。  
* 局所的に「鍛える」というよりも、必要なタイミングで勝手に使われるように「整える」という捉え方が重要になります 。  
* 肩甲骨の外側からついているイメージを持たれがちですが、実際は肩甲骨の内側（内側縁）から肋骨にかけてついています 。  
* 肩甲骨を前側に引っ張る「外転」の働きに強く関与します 。  
* 肩甲骨は関節としての構造的要素が少なく基本的には肋骨にひっついているだけであり、そのひっつける役割を前鋸筋が担っています 。  
* 前鋸筋が正しく働かないと、四つん這いになった際に肩甲骨が浮き出る「翼状肩甲」と呼ばれる状態になります 。

## **上部繊維と下部繊維の働き**

* 前鋸筋は大きく上部繊維と下部繊維に分けられます 。  
* 上部繊維は、肩甲骨の外転や、ラットプルダウンなどで肩を下げる「下制」の運動に使われます 。  
* 下部繊維は、腕を上げる動作の際に肩甲骨の下部を前方向に引っ張る「上方回旋」や、肩甲骨の「後傾」に関わります 。  
* 手が上にある状態、横にある状態、下にある状態など、どの位置で前鋸筋（上部・下部のどちらか）がうまく機能していないかを見分けることが改善には必要です 。

## **僧帽筋との関連と肩甲上腕リズム**

* 前鋸筋は僧帽筋と強い関連性を持ちます 。  
* 上部繊維は僧帽筋の中部・下部と共同して働き、下部繊維は僧帽筋の上部・下部と共同して働きます 。  
* 肩甲骨と上腕が2対1の割合で動く「肩甲上腕リズム」を正しく行うには、肩甲骨の動きが主導となる必要があります 。  
* 前鋸筋と僧帽筋が同時に収縮（共収縮）していないと、骨のセントレーション（中心位）が取れず、動きに偏りが出てしまいます 。

## **トレーニングにおける重要性（プル動作とプッシュ動作）**

* 前鋸筋はベンチプレスなどのプッシュ動作で注目されがちですが、ローイングなどのプル（引く）動作においても非常に重要です 。  
* 引く動作の際、背中の筋肉の出力方向に対して肋骨が過剰に外旋しないよう、前鋸筋が前方向にストレスをかけることで背中が十分な収縮を起こせます 。  
* 前鋸筋が働くことで肩がしっかりと落ち、脇に腕が近づいて背中に効きやすくなります 。  
* 広背筋などのアウターマッスルが強く入りすぎると、肩の抜け感や不安定感に繋がるため、拮抗する前鋸筋や僧帽筋の働きでブレーキをかける必要があります 。  
* プッシュアップのポジションでは、肩甲骨の外転だけでなく、内転の動作（前鋸筋がストレッチされること）が出るかも非常に重要です 。

## **腹筋（外腹斜筋）や呼吸との連動**

* 筋膜の連鎖において前鋸筋は外腹斜筋と連結しているため、前鋸筋が活動しないと外腹斜筋も効かなくなります 。  
* 外腹斜筋が効かないと肋骨が内旋しなくなり、リブフレア（肋骨が開いた状態）の原因となります 。  
* 腕を上げる動作の際、頭は「腕を上げる」という指令しか出していませんが、姿勢反射（先行随伴性姿勢調節など）によって無意識に同側の腹斜筋やハムストリングスが収縮し、姿勢を保ちます 。  
* 不安定な人や腹筋に力が入りにくい人は、プランクなどのCKC（クローズド・キネティック・チェーン）ポジションで負荷をかけたまま呼吸の練習をするのがおすすめです 。

## **ボディメイクと運動学習への影響**

* 前鋸筋が働くことで体幹に対して反作用が働き、遠い位置から腕がしっかり戻ってくるため、筋肉1個1個の丸みが出やすくなります 。  
* 前鋸筋が働かないと腕が遠い位置で止まってしまい、胸などの収縮が不十分になります 。  
* ピラティスは、前方の弱い前鋸筋や外腹斜筋、骨盤の後傾など、インナーや連動性に注力してデザインされています 。  
* ピラティスのような「ソフトウェア」の部分（連動性など）を獲得するだけでなく、それを日常や「ハードウェア」の部分（アウターマッスル）に転移させて固めていくことで、壊れにくい体のシステムが出来上がります 。
