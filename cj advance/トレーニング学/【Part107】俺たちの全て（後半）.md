

# **【Part107】俺たちの全て（後半）**

## **1\. 内的運動（ソフト）と外的運動（ハード）の役割と統合**

* 過剰にハードなトレーニングばかりを行うと怪我のリスクが高まる一方で、ピラティスなどの内的運動（ソフトな部分）だけに偏るのも不十分であり、両方の要素を混在させることが重要です 。  
* ソフトな部分は身体の全体の配列を整えるのに影響しますが、骨格そのものを伸ばして固めたり、目に見える体型の変化をもたらすには、アウターマッスルへのハードな刺激が不可欠です 。  
* ハムストリングスや腹直筋などの筋肉が最終的に連結して綺麗に働くことが望ましく、それぞれの役割の違いを理解し、両方を大事にする必要があります 。  
* クライアントによっては、自分の内側に意識を向けすぎると、かえって動きが分からなくなり、身体が硬くなってしまうケースもあるため指導には注意が必要です 。

## **2\. 過度なトレーニングや栄養摂取が招く「老化・酸化」のリスク**

* マラソンなどのハードな運動を長年続けている選手は、酸化によって肌の劣化などが見られやすく、必ずしも健康的な見た目になるとは限りません 。  
* 筋トレ（無酸素運動）は有酸素運動に比べれば酸化の量は少なくなるものの、過剰な糖質摂取（血糖値の上げすぎ）や塩分の摂りすぎは、心臓の肥大など身体への大きな負担となります 。  
* ロイシンなどのサプリメントは筋合成を強力に高めますが、合成（人体としての進化・代謝）を人為的に早めることは、結果として身体を早く老けさせる（酸化や糖化を促進する）ことにも繋がります 。  
* 結論として、どのような運動や栄養摂取も「適度」が一番であり、過度な負荷は病気や老化のリスクを高めるなど、身体への尋常ではない負担となることを理解しておくべきです 。
