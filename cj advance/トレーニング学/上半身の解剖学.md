# 上半身の機能解剖学と動作分析ガイド

> **Category**: /sports/anatomy
> **Tags**: #Anatomy #UpperBody #Biomechanics #JointByJoint #ScapulohumeralRhythm

## なぜこれが必要か(目的・背景)

トレーナーとして、クライアントの身体の動きを正確に評価し、効果的かつ安全な指導を行うため。
特に上半身の関節運動（肩、脊柱等）の仕組みや、エラー動作（代償運動）のメカニズムを理解することで、怪我を防ぎパフォーマンスを最大化する「根拠に基づいた指導」を可能にする。

---

## 基本(解剖学的原則)

### 1. 動作の3平面

* **矢状面 (Sagittal)**: 人体を左右に分ける面。前後の動き（屈曲・伸展）。
* 例: ショルダープレス（屈曲）、ローイング（伸展）。


* **前額面 (Frontal)**: 人体を前後に分ける面。左右の動き（外転・内転）。
* 例: サイドレイズ（外転）、ラットプルダウン（内転）。


* **水平面 (Transverse)**: 人体を上下に分ける面。回旋の動き（内旋・外旋）。
* 例: フライ動作（水平内転）、回旋運動。



### 2. 肩甲上腕リズム (2:1の法則)

肩関節の動きは、上腕骨と肩甲骨の連動によって成立する。

* **比率**: 上腕骨の動き **2** に対し、肩甲骨の動き **1**。
* **例**: 肩を180°挙上する場合、上腕骨（肩甲上腕関節）が120°、肩甲骨（肩甲胸郭関節）が60°動く。
* **重要性**: このリズムが崩れると、インピンジメントや可動域制限の原因となる。

### 3. ジョイントバイジョイント (Joint-by-Joint)

関節は「可動性 (Mobility)」と「安定性 (Stability)」の役割を交互に持つ。

| 関節 | 役割 | 機能低下時のリスク |
| --- | --- | --- |
| **肩関節** | **Mobility (可動)** | 可動域制限、痛み |
| **肩甲胸郭** | **Stability (安定)** | 肩こり、首の痛み、パワー伝達ロス |
| **胸椎** | **Mobility (可動)** | 腰椎への過剰負荷、呼吸制限 |
| **腰椎** | **Stability (安定)** | 腰痛、ヘルニア |

## 実践：代償運動の特定と修正

### エラー動作のメカニズム

ある関節の機能が低下すると、隣接する関節が無理に動いて補おうとする（代償運動）。

* **例1: ベンチプレスで腰が反りすぎる**
* **原因**: 胸椎の伸展可動域不足、または肩関節の屈曲制限。
* **代償**: 可動性不足を補うため、安定性が必要な「腰椎」を過剰に伸展させてしまう。
* **対策**: 胸椎のモビリティエクササイズ、広背筋のリリース。


* **例2: ラットプルダウンで肩がすくむ**
* **原因**: 肩甲骨の下制機能（安定性）不足。
* **代償**: 僧帽筋上部を使って肩甲骨を無理に引き上げようとする。
* **対策**: 前鋸筋や僧帽筋下部の活性化、パッキングの習得。



### 腹圧 (IAP) の活用

代償運動を防ぐ最大の鍵は「体幹の安定性」にある。

* **基本**: 呼吸（特に呼気）によって横隔膜と骨盤底筋を連動させ、腹腔内圧を高める。
* **指導**: 「息を吐ききって肋骨を閉じる（リブフレア防止）」ことで、脊柱の安定性を確保してから動作を開始させる。

## 現場メモ(指導のポイント)

* **「整えてから鍛える」**: 可動域制限や安定性不足がある状態で高重量を扱わせない。まずはモビリティドリルやコレクティブエクササイズで機能を正常化する。
* **肩甲骨面 (Scapular Plane)**: 肩甲骨は前方に約30°傾いている。プレス系種目は真横（0°）ではなく、少し前方（30°）で行う方が関節負担が少ない。

## 結論

トレーニング指導の質は、「なぜそのフォームになるのか」を解剖学的に説明できるかで決まる。
**「ジョイントバイジョイント」**と**「腹圧」**の概念を軸に、クライアントの代償運動を見抜き、根本原因（可動域不足or安定性不足）にアプローチすることで、怪我なく最短で成果を出すことができる。

以下原文
複数の資料は、トレーナー向けに**上半身および下半身**における**解剖学とバイオメカニクス**の知識を体系的に提供する研修資料および解説動画の文字起こしで構成されています。内容は、肩関節、脊柱、股関節、膝関節など、各主要な関節における屈曲、伸展、回旋といった基本的な動きと、それに関与する具体的な筋肉群や**関節可動域**を詳細に示しています。特に重要な概念として、肩の円滑な動作に不可欠な**肩甲上腕リズム（2:1の比率）や、関節の安定性と可動性を管理するジョイントバイジョイントアプローチ**が取り上げられています。また、トレーニングの指導を深めるため、誤った動作パターンである**代償運動**が発生する原因を特定し、**腹圧**の調整や適切なエクササイズを通じてこれを修正するための具体的なアプローチ方法も解説されています。

三つの基本面

* 前額面: 人体を前後に分ける面。内転、外転など左右の動きがこの面上で起こる。  
* 矢状面: 人体を左右に分ける面。屈曲、伸展など前後の動きがこの面上で起こる。  
* 水平面: 人体を輪切りにして上下に分ける面。回旋の動作がこの面上で起こる。

上半身の関節と動き

* 肩関節:  
* 屈曲 (矢状面): 腕を前から上げる (180°)。三角筋前部、大胸筋上部、烏口腕筋、上腕二頭筋が関与。  
* 伸展 (矢状面): 腕を後方に引く (60°)。広背筋、上腕三頭筋長頭、大円筋、三角筋後部、小円筋が関与。  
* 外転 (前額面): 腕を外側から上げる (180°)。三角筋中部、棘上筋が関与。  
* 内転 (前額面): 腕を閉じる (0°)。広背筋、大胸筋下部、大円筋が関与。  
* 内旋 (水平面): 前へならえから内側に回す (80°)。肩甲下筋、大胸筋、広背筋、大円筋が関与。  
* 外旋 (水平面): 前へならえから外側に回す (60°)。棘下筋、小円筋が関与。  
* 水平屈曲（水平内転） (水平面): 肩関節90°外転位から腕を前方に移動 (135°)。大胸筋、三角筋前部が関与。  
* 水平伸展（水平外転） (水平面): 肩関節90°外転位から腕を後方に移動 (30°)。三角筋後部、棘下筋、小円筋が関与。  
* 肩甲帯: (動きは主に前額面)  
* 内転: 肩甲骨を内側に寄せる (20°)。僧帽筋、菱形筋が関与。  
* 外転: 肩甲骨を外に開く (20°)。前鋸筋、小胸筋が関与。  
* 挙上: 肩甲骨を上げる (肩をすくめる) (20°)。僧帽筋上部、肩甲挙筋、菱形筋が関与。  
* 下制: 肩甲骨を下げる (10°)。僧帽筋下部、小胸筋が関与。  
* 上方回旋: 肩甲骨の下角が上を向く。肩関節外転と連動（肩甲上腕リズム）(肩関節60°外転で肩甲骨30°上方回旋)。前鋸筋下部、僧帽筋上部が関与。  
* 下方回旋: 肩甲骨の下角が下を向く (60°)。菱形筋、肩甲挙筋、小胸筋が関与。  
* 肘関節: (動きは主に矢状面)  
* 屈曲: 肘を曲げる (145°)。上腕二頭筋、上腕筋、腕橈側筋が関与。  
* 伸展: 肘を伸ばす (5°)。上腕三頭筋、肘筋が関与。  
* 前腕: (動きの面は記載なし)  
* 回外: 肘関節90°屈曲位で掌を上に向ける (90°)。上腕二頭筋、回外筋が関与。  
* 回内: 肘関節90°屈曲位で掌を下に向ける (90°)。円回内筋、方形回内筋が関与。  
* 手関節:  
* 掌屈 (矢状面): 手首を掌側に曲げる (90°)。橈側手根屈筋、尺側手根屈筋が関与。  
* 背屈 (矢状面): 手首を手の甲側に曲げる (70°)。総指伸筋、長橈側手根伸筋、短橈側手根伸筋、尺側手根伸筋が関与。  
* 橈屈 (前額面): 手首を橈骨側に曲げる (25°)。橈側手根屈筋、長・短橈側手根伸筋、長母指外転筋が関与。  
* 尺屈 (前額面): 手首を尺骨側に曲げる (55°)。尺側手根屈筋、尺側手根伸筋が関与。  
* 脊柱:  
* 頚椎 (C1-C7): 屈曲60°、伸展50°、回旋45-50°  
* 胸椎 (T1-T12): 屈曲40°、伸展25°、回旋30-35° (回旋が得意)  
* 腰椎 (L1-L5): 屈曲50°、伸展15°、回旋5-15° (屈曲・伸展が得意、回旋が苦手)  
* 仙骨 (S1-S5): うなずき（ニューテーション）、起き上がり（カウンターニューテーション）

肩甲上腕リズム

* 上腕骨と肩甲骨の動きの比率。  
* 上腕骨と肩甲骨は2：1の割合で動く。  
* 肩を60°外転すると、肩甲上腕関節は40°、肩甲胸郭関節は20°上方回旋する。

肩甲骨面

* 肩甲骨は背面にあるが、肋骨の丸みに沿って、水平面から前方へ約30°の角度を持つ。  
* この位置での動作は関節への負担が少ない。

ジョイントバイジョイント

* 関節をMobility Joint（可動性）とStability Joint（安定性）に分類する考え方。  
* 上半身の例：上部頚椎 (M)、下部頚椎 (S)、肩関節 (M)、肩甲胸郭関節 (S/M)、胸椎 (M)、腰椎骨盤帯 (S)。  
* ある関節のMobilityが低下すると、隣接するStability Jointが代償的に動き、代償動作を引き起こす可能性がある。

腹圧（腹腔内圧 \- IAP）

* 腹部内の空気圧で、脊柱の安定に最も重要。  
* 呼吸により機能が最大化される。  
* 適切な呼吸がなければ、他の動作パターンも適切にならない。  
* 腹圧を最大化するための要素：長く吐く (Exhale)、上下均等 (Hi-Lo)、前後均等 (CORE-Expansion)、腹腔内圧 (IAP)。  
* 主な関与筋：横隔膜、腹横筋、多裂筋、骨盤底筋群。  
* 横隔膜は呼吸とスタビリティの「Dual Function」を持つ。

代償運動とそのアプローチ

* 代償運動は、特定の関節や筋肉の機能不全を他の部位が補う動き。  
* 例：股関節屈曲時の腰椎屈曲、肩関節屈曲時の腰椎過伸展。  
* アプローチ方法：  
* 動作の確認（正しいフォームか）。  
* 代償運動の見極め（フォーム、反応）。  
* 原因部位へのアプローチ（ストレッチ、リリース）。  
* 不活性筋の活性化。  
* 動作の学習。  
* 代償運動の例：  
* ベンチプレス：腰椎伸展によるアーチ。原因：頭部前方変位、肩内巻き、骨盤過前傾など。  
* ラットプルダウン：肩甲骨挙上、胸椎後湾。原因：スウェイバックなど。  
* サイドレイズ：シュラッグ動作、肩甲骨前傾。原因：特に姿勢不良はないが、肩関節屈曲・外転時に肩挙上しやすい。

小テスト解答集

* 水平面です。水平面は人体を輪切りにして上下に分ける面であり、この面上で回旋動作が行われます。  
* 三角筋中部と棘上筋です。これらの筋肉が共同して腕を体側から外側に挙げる動きを担います。  
* 上腕骨と肩甲骨の動きの比率を指します。肩の挙上などの際に、これら二つの骨が協調して動くパターンです。  
* 肩甲上腕関節は40°、肩甲胸郭関節は20°上方回旋します。これは肩甲上腕リズムの2：1の比率に基づいています。  
* 水平面から約30°前方です。この角度により、上腕骨頭と関節窩の適合が最適化され、関節負担が軽減されます。  
* 可動性（動き）が要求される関節を指します。これは、ある関節の動きやすさが隣接関節の安定性に影響を与えるというジョイントバイジョイントの概念の一部です。  
* 腹腔内圧は腹部内の空気圧であり、これが高まることで体幹の中心部が安定し、脊柱への負担を軽減するからです。特に重いものを持ち上げる際などに重要になります。  
* 横隔膜と腹横筋、あるいは多裂筋や骨盤底筋群などが挙げられます。これらの筋肉が連携して腹腔内圧を高め、体幹の安定化に寄与します。  
* 上腕二頭筋、上腕筋、腕橈側筋です。これらの筋肉の収縮によって肘関節が曲がる（屈曲する）動作が起こります。  
* 腰椎は屈曲と伸展が得意ですが、回旋が苦手です。一方、胸椎は回旋が得意で、屈曲と伸展は腰椎に比べて苦手です。

用語集

* 前額面 (Frontal Plane): 人体を前後に分ける仮想的な面。内転や外転といった左右方向の運動がこの面上で起こる。  
* 矢状面 (Sagittal Plane): 人体を左右に分ける仮想的な面。屈曲や伸展といった前後方向の運動がこの面上で起こる。  
* 水平面 (Transverse Plane): 人体を上下に分ける仮想的な面。回旋といった回転運動がこの面上で起こる。  
* 屈曲 (Flexion): 関節の角度が小さくなる動き（一般的に、矢状面での体の前面方向への動き）。  
* 伸展 (Extension): 関節の角度が大きくなる動き（一般的に、矢状面での体の後面方向への動き）。  
* 内転 (Adduction): 体の中心線に向かって体を閉じる動き（一般的に、前額面での運動）。  
* 外転 (Abduction): 体の中心線から離れるように体を開く動き（一般的に、前額面での運動）。  
* 内旋 (Internal Rotation): 体の中心線に向かって回転する動き（一般的に、水平面での運動）。  
* 外旋 (External Rotation): 体の中心線から離れるように回転する動き（一般的に、水平面での運動）。  
* 水平屈曲 / 水平内転 (Horizontal Flexion / Horizontal Adduction): 肩関節が約90°外転位から水平面で体の中央に向かって腕を動かすこと。  
* 水平伸展 / 水平外転 (Horizontal Extension / Horizontal Abduction): 肩関節が約90°外転位から水平面で体の中央から離れるように腕を動かすこと。  
* 挙上 (Elevation): 肩甲骨などが上方に移動する動き。  
* 下制 (Depression): 肩甲骨などが下方に移動する動き。  
* 上方回旋 (Upward Rotation): 肩甲骨の下角が外側かつ上方に移動するような回転運動。  
* 下方回旋 (Downward Rotation): 肩甲骨の下角が内側かつ下方に移動するような回転運動。  
* 掌屈 (Palmar Flexion): 手関節が掌側に曲がる動き。  
* 背屈 (Dorsiflexion): 足関節または手関節が背側に曲がる動き。  
* 橈屈 (Radial Deviation): 手関節が橈骨側（親指側）に曲がる動き。  
* 尺屈 (Ulnar Deviation): 手関節が尺骨側（小指側）に曲がる動き。  
* 回外 (Supination): 前腕が回旋して、掌が上を向く動き（または足部が外側にねじれる動き）。  
* 回内 (Pronation): 前腕が回旋して、掌が下を向く動き（または足部が内側にねじれる動き）。  
* 肩甲上腕リズム (Scapulohumeral Rhythm): 上腕骨と肩甲骨の協調的な動きの比率。  
* 肩甲骨面 (Scapular Plane): 肋骨の湾曲に沿った肩甲骨の自然な向き。水平面から前方へ約30°の角度を持つとされる。  
* ジョイントバイジョイント (Joint by Joint Approach): 関節をMobility Joint（可動性）とStability Joint（安定性）に分類し、それぞれに求められる機能があるという考え方。  
* Mobility Joint: 可動性が重要視される関節。  
* Stability Joint: 安定性が重要視される関節。  
* 腹腔内圧 (Intra-abdominal Pressure \- IAP): 腹腔内の圧力。体幹の安定性に重要。  
* 代償運動 (Compensatory Movement): ある関節や筋肉の機能不全を他の部位が補うために起こる、理想的ではない動作パターン。  
* ニューテーション (Nutatiton): 仙骨が前方に傾く動き（うなずき）。  
* カウンターニューテーション (Counter-nutation): 仙骨が後方に傾く動き（起き上がり）。

### Q1: 上半身の解剖学研修の主な目的は何ですか？

A1: この研修の主な目的は、トレーナーが上半身の解剖学への理解を深め、その知識に基づいたより効果的な指導を行うことです。具体的には、肩関節、肩甲帯、肘関節、前腕、手関節、そして脊柱といった上半身の主要な部位の解剖学、それぞれの関節の基本的な動き（屈曲、伸展、外転、内転、回旋など）とそれに Primary に関わる筋肉、そしてそれぞれの関節の可動域について学びます。また、肩甲上腕リズムや肩甲骨面といった概念、そして解剖学に基づいたトレーニング方法についても習得することを目指します。

### Q2: 研修を受ける上での重要な注意事項は何ですか？

A2: 研修の効果を最大化するために、いくつかの重要な注意事項があります。まず、研修中は積極的にノートを取り、後から自分が理解しやすいように整理することが推奨されます。次に、わからないことがあればその場で質問し、わからないまま先に進まないように自己主導的な学習姿勢を持つことが求められます。また、定期的に質問がないか聞かれた際には、必ず「ないです」と反応することが重要です。最後に、研修で学んだ内容は必ず現場や他の人にアウトプットすることで、最も効果的なインプットに繋がるとされています。

### Q3: 人体の基本的な動きを説明する三つの基本面とは何ですか？

A3: 人体の基本的な動きは、三つの基本面で説明されます。一つ目は「前額面」で、人体を前後に分けた面です。内転や外転といった左右の動きはこの面上の動きとなります。二つ目は「矢状面」で、人体を左右に分けた面です。屈曲や伸展といった前後の動きはこの面上の動きとなります。三つ目は「水平面」で、人体を輪切りにして上下に分けた面です。回旋の動作はこの面上の動きとなります。

### Q4: 肩関節の動きとその可動域、そして主な関与筋について教えてください。

A4: 肩関節は非常に可動域の広い関節で、様々な動きが可能です。主な動きとその概要、可動域、関与筋は以下の通りです。

* 屈曲: 腕を前から上げる動作。矢状面上。可動域は約180°。三角筋前部、大胸筋上部、烏口腕筋、上腕二頭筋などが関与します。  
* 伸展: 腕を後方に引く動作。矢状面上。可動域は約60°。広背筋、上腕三頭筋長頭、大円筋、三角筋後部、小円筋などが関与します。  
* 外転: 腕を外側から上げる動作。前額面上。可動域は約180°。三角筋中部、棘上筋などが関与します。  
* 内転: 腕を閉じる動作。前額面上。可動域は約0°。広背筋、大胸筋下部、大円筋などが関与します。  
* 内旋: 小さく前へならえした位置から内側に回す動作。水平面上。可動域は約80°。肩甲下筋、大胸筋、広背筋、大円筋などが関与します。  
* 外旋: 小さく前へならえした位置から外側に回す動作。水平面上。可動域は約60°。棘下筋、小円筋などが関与します。  
* 水平屈曲（水平内転）: 肩関節90°外転位で上腕を水平面で前方に移動させる動き。水平面上。可動域は約135°。大胸筋、三角筋前部などが関与します。  
* 水平伸展（水平外転）: 肩関節90°外転位で上腕を水平面で後方に移動させる動き。水平面上。可動域は約30°。三角筋後部、棘下筋、小円筋などが関与します。

### Q5: 肩甲上腕リズムとは何ですか？

A5: 肩甲上腕リズムとは、肩関節の外転や屈曲といった腕の動きにおいて、上腕骨と肩甲骨が連動して動く比率のことを指します。具体的には、肩関節が外転または屈曲する際に、上腕骨と肩甲骨は約2：1の割合で動くとされています。例えば、肩を60°外転すると、肩甲上腕関節で40°、肩甲胸郭関節（肩甲骨の動き）で20°の上方回旋が起こります。このリズムが正常に機能することで、肩関節への負担を軽減し、スムーズな動きを可能にしています。

### Q6: ジョイントバイジョイントのコンセプトと代償運動における考え方について教えてください。

A6: ジョイントバイジョイントは、身体の関節を可動性（Mobility）が求められる関節と安定性（Stability）が求められる関節に分けて考えるコンセプトです。例えば、上部頸椎、肩関節、胸椎、股関節、足関節は主に可動性が求められ、下部頸椎、肩甲胸郭関節、腰椎骨盤帯、膝関節、足部は主に安定性が求められます（ただし、一部の関節は両方の側面を持ちます）。  
このコンセプトは代償運動を理解する上で重要です。例えば、股関節の可動性が低下している場合、その不足を補うために腰椎が過剰に屈曲するといった代償運動が発生することがあります。同様に、肩甲骨や胸椎、胸郭の可動性が低下すると、肩関節の屈曲時に腰椎が過伸展するといった代償運動が見られます。代償運動に対してアプローチする際には、代償運動が起きている原因となる部位（可動性の低下した関節や安定性の低下した関節）を特定し、その機能改善のためのエクササイズを行うことが重要であると考えられています。これは「整えてから鍛える」という考え方に基づいています。

### Q7: 腹腔内圧（IAP）はトレーニングにおいてどのような役割を果たしますか？

A7: 腹腔内圧（IAP）は、腹部内の空気圧を指し、脊柱を安定させる上で最も重要な要素の一つです。適切な腹腔内圧を維持することは、体幹の安定性を高め、怪我の予防やパフォーマンス向上に繋がります。IAPの機能は、呼吸によって最大化されます。特に、横隔膜、腹横筋、多裂筋、骨盤底筋群といったコアマッスルが適切に機能することで、効率的な呼吸と高い腹腔内圧が実現されます。IAPを最大化するためには、長く息を吐く（Exhale）、上下均等に呼吸する（Hi-Lo）、腹部全体が前後均等に膨らむように呼吸する（CORE-Expansion）といった要素が重要とされています。横隔膜は呼吸だけでなくスタビリティという二つの機能を持っており、この「Dual Function」が適切に働くことが理想です。

### Q8: トレーニング中によく見られる代償運動の例とそのアプローチ方法について教えてください。

A8: トレーニング中には、本来の動きを補うために意図しない別の部位が過剰に動く「代償運動」がよく見られます。いくつかの例とそのアプローチ方法は以下の通りです。

* ベンチプレスにおける腰椎の伸展（過度なアーチ）: これは、頭部前方変位、肩の内巻き、骨盤の前傾といった姿勢不良や、肩甲骨・胸椎・胸郭の可動性低下、または腰椎・骨盤帯の安定性低下が原因で起こることがあります。アプローチとしては、原因となっている部位（筋緊張や短縮、動き方の学習不足など）を特定し、ストレッチやリリースによるアプローチ、不活性筋の活性化、そして動作の学習を行います。  
* ラットプルダウンにおける肩（肩甲骨）の挙上や胸椎の後弯: これは、肩甲骨・胸椎・胸郭の可動性低下、または腰椎・骨盤帯の安定性低下が原因で起こることがあります。特にスウェイバック姿勢の方に見られることがあります。アプローチはベンチプレスの場合と同様に、原因部位へのアプローチと動作学習を行います。  
* サイドレイズにおけるシュラッグ動作や肩甲骨の前傾: これは、特に姿勢不良がない場合でも、肩関節の屈曲や外転時に肩の挙上も伴いやすい方に多く見られます。アプローチは原因の特定と改善に向けたエクササイズ、そして正しい動作パターンの習得を目指します。

代償運動へのアプローチの一般的な流れは、まず動作の確認（正しいフォームでできているか）、次に代償運動の見極め（フォームや顧客の反応）、そして原因部位へのアプローチ（筋緊張や短縮に対するストレッチやリリース、または動き方の学習不足に対する動作指導）、不活性筋の活性化、最後に動作の再学習となります。

ご質問ありがとうございます。ジョイントバイジョイント（Joint-by-Joint Approach）の原則に基づいた各関節の分類について、ソースに基づき説明します。

ジョイントバイジョイントアプローチでは、人体は\*\*可動性（Mobility Joint）**と**安定性（Stability Joint）\*\*の機能を持つ関節が交互に並んでいるという考え方に基づいています。

肩甲上腕リズムについて、その比率と具体的な動きを以下の通り説明します。

### **1\. 肩甲上腕リズムの定義**

**肩甲上腕リズム**とは、**上腕骨と肩甲骨の動きの比率**のことを言います。

これは、肩関節複合体の動作時における、**肩甲上腕関節**（上腕骨と肩甲骨の関節）と**肩甲胸郭関節**（肩甲骨と胸郭の間の機能的な関節）の協調的な動きを示します。

### **2\. 比率と具体的な動き**

上腕骨と肩甲骨は、一般的に**2：1の割合**で動きます。この比率は「**肩甲上腕関節：肩甲胸郭関節 ＝ 2：1**」で示されます。

具体的な肩の外転（腕を外側から上げていく動作）における動きの比率は以下の通りです。

| 動作 | 関節 | 角度 |
| ----- | ----- | ----- |
| **肩関節 60°外転** | 全体の動き | 60° |
|  | **肩甲上腕関節** | 40° |
|  | **肩甲胸郭関節** | 20°上方回旋する |

つまり、肩を60°外転する動作が行われた場合、そのうち40°の動きは上腕骨自体が肩甲骨に対して動き（肩甲上腕関節）、残りの20°の動きは肩甲骨が上方回旋することによって生じます（肩甲胸郭関節）。

なお、肩甲帯の解剖学の記載として、肩関節が60°外転すると、肩甲骨は30°上方回旋するという記述も別の箇所にあります。

**前額面**は**人体を前後に分けた面**であり、**内転**や**外転**のように**左右の動き**がこの面上に起こります。

腹圧と安定性は、脊柱や体幹を支持する上で最も重要であり、トレーニング中の代償運動の発生を防ぎ、是正するための鍵となります。

### **1\. 腹圧（IAP：腹腔内圧）の活用**

腹腔内圧（IAP）は、**腹部内の空気圧**のことを指し、**脊柱を安定させる最も重要なセクション**です。

**腹圧と安定性の関係:**

* **呼吸とIAP:** 呼吸（breathing）によりIAPの機能は最大化されます。呼吸が適正化されなければ、その他の動作パターンを適正化することはできません。  
* **横隔膜の二重機能（Dual Function）:** 横隔膜には「呼吸」と「スタビリティ（安定性）」という二つの機能があり、これらは常に一対であり、「Dual Function」でなければならないとされています。  
* **腹圧が入りづらい姿勢:** Rib Flare（肋骨が外に開いている状態）は、腹圧が入りづらい姿勢の一例です。

**IAPを最大化するための要素:** 腹圧をしっかりと入れるためには、以下の要素を意識します。

1. **Exhale（長く吐く）**  
2. **Hi-Lo（上下均等に）**  
3. **CORE-Expansion（前後均等に）**  
4. **IAP（腹腔内圧）**

IAPに主に作用する筋肉としては、**横隔膜**、**腹横筋**、**多裂筋**、**骨盤底筋群**が挙げられます。

### **2\. 安定性（Stability）に基づいた代償運動の原因特定**

安定性の機能は、「ジョイント・バイ・ジョイント（Joint-by-Joint）」の考え方を用いて、代償運動の原因特定に活用されます。

**ジョイント・バイ・ジョイントの原則:** 人体は、可動性（Mobility Joint）と安定性（Stability Joint）の機能を持つ関節が交互に並んでいるという考え方です。

| 関節の種類 | 機能 |
| ----- | ----- |
| 上部頸椎 | Mobility Joint（可動性） |
| **下部頸椎** | **Stability Joint（安定性）** |
| 肩関節 | Mobility Joint（可動性） |
| 肩甲胸郭関節 | Stability Joint / Mobility Joint |
| 胸椎 | Mobility Joint（可動性） |
| **腰椎骨盤帯** | **Stability Joint（安定性）** |
| 股関節 | Mobility Joint / Stability Joint |
| **膝関節** | **Stability Joint（安定性）** |
| 足関節 | Mobility Joint（可動性） |
| 足部 | Stability Joint（安定性） |

**安定性低下が引き起こす代償運動の例:** 特に安定性が求められる**腰椎・骨盤帯**の機能低下は、様々な代償運動の原因となります。原因を特定する際は、*可動性の低下*によるものか、*安定性の低下*によるものかを区別することが重要です。

| 動作 | 代償運動 | 安定性の原因特定 | 是正のアプローチ |
| ----- | ----- | ----- | ----- |
| 股関節屈曲 | **腰椎の屈曲** | **腰椎・骨盤帯の安定性低下** | **腰椎・骨盤帯の安定性を改善するエクササイズが必要** |
| 肩関節屈曲 | **腰椎の過伸展** | **腰椎・骨盤帯の安定性低下** | **腰椎・骨盤帯の安定性を改善するエクササイズが必要** |
| ベンチプレス | **腰椎の伸展でアーチ** | 腹圧と体幹の安定性不足 | 安定性改善エクササイズ |

### **3\. 代償運動に対する是正アプローチ**

代償運動の是正において重要なのは、\*\*「整えてから鍛える」\*\*という原則です。安定性や可動性の問題を解決してからトレーニングを行うことが大切です。

代償動作に対するアプローチ方法は、以下の手順を踏みます。

1. **動作の確認:** 正しいトレーニングフォームができているかを確認します。  
2. **代償運動の見極め:** 動作フォームやお客様の反応から、代償運動が発生していることを見極めます。  
   * この際、**「筋の緊張や短縮によって動かないのか」**、\*\*「動かし方がわからないのか」\*\*を見極めます。  
3. **代償運動が起きている原因となる部位に対してアプローチ:** 原因部位に対してストレッチやリリースなどのアプローチを行います。  
   * もし原因が「腰椎・骨盤帯の安定性低下」であれば、安定性を改善するエクササイズを実施します。  
4. **不活性の筋肉を活性化**。  
5. **動作の学習**。

このように、**腹圧**（IAP）の確保を通じて**脊柱の安定性**を最大化すること、そして**ジョイント・バイ・ジョイント**の原則に基づき、代償運動の原因が**安定性**の欠如にあるのかを特定すること が、代償運動の是正とトレーニング効果の最大化につながります。

---

安定性の機能活用は、基礎工事のようなものです。どれだけ高層ビル（高いパフォーマンス）を建てようとしても、土台（腰椎・骨盤帯の安定性）が不安定であれば、その上の構造物（四肢の動きや力の発揮）は歪み（代償運動）を生じ、最終的には建物の崩壊（怪我や機能不全）につながります。腹圧を高めることは、この土台を強固に固めるためのセメントのような役割を果たします。

ご質問ありがとうございます。肩甲上腕リズムとジョイントバイジョイントの原則は、身体の各関節が連携し、効率的かつ負担の少ない動作を実現するために重要な役割を果たします。

以下に、ソースに基づき、それぞれの概念が最適な動作機能にどのように寄与するかを説明します。

---

### **1\. 肩甲上腕リズム（Scapulohumeral Rhythm）の寄与**

**肩甲上腕リズム**とは、**上腕骨と肩甲骨の動きの比率**のことを指します。このリズムは、肩関節複合体の円滑な動作と関節への負担軽減に不可欠な役割を果たします。

#### **定義と比率**

* 上腕骨と肩甲骨は**2：1の割合**で動きます。  
* 具体的には、肩を60°外転させる動作において、**肩甲上腕関節**が40°、**肩甲胸郭関節**が20°の**上方回旋**を行います。  
* また、別の記載として、肩関節が60°外転すると、肩甲骨は30°上方回旋することが示されています。

#### **最適な動作機能への寄与**

この協調的なリズムが守られることにより、肩関節は広い可動域（例：屈曲180°、外転180°）を達成できます。

さらに、肩甲上腕リズムと関連の深い**肩甲骨面**の理解は、関節の負担を軽減します。

* **肩甲骨面**とは、丸みのある肋骨の上に位置する肩甲骨が、水平面に対して前方に**30°の角度**を持っている位置を指します。  
* この肩甲骨面での動作は、**上腕骨頭と関節窩の適合が最適化**され、結果として**関節への負担が少なくなる**とされています。

このように、肩甲上腕リズムは、肩の動作時に上腕骨と肩甲骨が正しい比率で動くことを保証し、関節の安定性と可動性を両立させることで、最適な動作機能に寄与します。

---

### **2\. ジョイントバイジョイント（Joint-by-Joint Approach）の寄与**

**ジョイントバイジョイント**アプローチは、身体の関節を\*\*可動性（Mobility Joint）**と**安定性（Stability Joint）\*\*のどちらを主として担うかによって分類し、動作の質を高めるための指導の基礎となる考え方です。

#### **関節の役割分担**

ジョイントバイジョイントの考え方では、各関節が可動性または安定性の役割を交互に担うとしています。

| 役割 | 関節の例 |
| ----- | ----- |
| **Mobility Joint**（可動性） | 上部頸椎、**肩関節**、胸椎、足関節 |
| **Stability Joint**（安定性） | 下部頸椎、**肩甲胸郭関節**（可動性の側面もある）、**腰椎骨盤帯**、膝関節、足部 |
| 両方の役割を持つ可能性 | 股関節（Mobility Joint / Stability Joint） |

#### **最適な動作機能への寄与（代償運動の回避）**

このアプローチの最大の寄与は、**代償運動**が発生した際に、その原因となる部位を特定し、適切なアプローチを行う指針を提供することです。最適な動作機能とは、本来動くべき関節が動き（Mobility）、安定すべき関節が安定している状態です。

1. **代償運動の原因特定:**

   * 例えば、股関節の**可動性**が低下している場合、その代償として腰椎の屈曲が起きる可能性があります。  
   * あるいは、腰椎・骨盤帯の**安定性**が低下しているために、腰椎の屈曲が起きる可能性もあります。  
2. **適切なアプローチ:**

   * 肩関節の屈曲時に腰椎の過伸展（反り）が代償運動として起きる場合、原因として**肩甲骨・胸椎・胸郭の可動性低下**、または**腰椎・骨盤帯の安定性低下**が考えられます。  
   * 最適な動作機能を取り戻すためには、原因となっている部位の可動性または安定性を改善するエクササイズが必要です。

このアプローチの重要な原則は、\*\*「整えてから鍛えること」\*\*であり、関節の役割に応じて可動性と安定性を改善することで、代償動作を防ぎ、動作の効率と安全性を高めることができます。

---

**まとめの比喩:**

肩甲上腕リズムは、オーケストラにおけるヴァイオリン（上腕骨）と指揮者（肩甲骨）の協奏曲の楽譜のようなものです。両者が定められた2:1の比率で正確に動くことで、肩という楽器は最高の音域（可動域）を奏で、関節に負担なく最高のパフォーマンス（最適な動作機能）を発揮できます。

一方、ジョイントバイジョイントの原則は、建物の設計図に例えられます。土台（Stability Joint）がしっかり安定し、窓やドア（Mobility Joint）がスムーズに開閉できるように、各部位がその役割を果たすことで、建物全体（身体）が崩れることなく、最高の機能性（最適な動作機能）を維持できるのです。もし窓の動きが悪くなれば（可動性低下）、無理に力を入れた別の壁（代償部位）にひずみが生じてしまいます。

三つの基本面（前額面、矢状面、水平面）は、上半身の各関節運動を分類し、その運動が人体をどのように分割する面上で起こるかを示す、解剖学的な基準となります。

これらの基本面と上半身の主要な関節運動との関連は以下の通りです。

### **1\. 矢状面（Sagittal Plane）**

**矢状面は人体を左右に分ける面**であり、**屈曲や伸展**のように**前後の動き**がこの面上に起こります。

| 関節 | 運動 | 動作の例 |
| ----- | ----- | ----- |
| **肩関節** | 屈曲 | 腕を前から上げていく動作 |
|  | 伸展 | 腕を後方に引く動作 |
| **肘関節** | 屈曲 | 肘を曲げる動作 |
|  | 伸展 | 肘を伸ばす動作 |
| **手関節** | 掌屈 | 手首を掌側に曲げる動き |
|  | 背屈 | 手首を手の甲側に曲げる動き |
| **脊柱** | 屈曲 | 首を前に倒す動き（頸椎）、胸を丸める動き（胸椎）、腰を曲げる動き（腰椎） |
|  | 伸展 | 首を後ろに倒す動き（頸椎）、胸を張る動き（胸椎）、腰を反る動き（腰椎） |

---

### **2\. 前額面（Frontal Plane）**

**前額面は人体を前後に分ける面**であり、**内転や外転**のように**左右の動き**がこの面上に起こります。

| 関節 | 運動 | 動作の例 |
| ----- | ----- | ----- |
| **肩関節** | 外転 | 腕を外側から上げていく動作 |
|  | 内転 | 腕を閉じる動作 |
| **肩甲帯** | 内転 | 肩甲骨を内側に寄せる動作 |
|  | 外転 | 肩甲骨を外に開く動作 |
|  | 挙上 | 肩甲骨を上げる動作（肩を竦める動作） |
|  | 下制 | 肩甲骨を下げる動作 |
|  | 上方回旋 | 肩甲骨の下角が上方に動く |
|  | 下方回旋 | 肩甲骨の下角が下方に動く |
| **手関節** | 橈屈 | 手首を橈骨側に曲げる動き |
|  | 尺屈 | 手首を尺骨側に曲げる動き |
| **脊柱** | 側屈 | 首を横に倒す動き（頸椎） |

---

### **3\. 水平面（Horizontal Plane）**

**水平面は人体を輪切りにして上下に分ける面**であり、**回旋の動作**がこの面上に起こります。

| 関節 | 運動 | 動作の例 |
| ----- | ----- | ----- |
| **肩関節** | 内旋 | 小さく前へならえをした位置から内側に回す動作 |
|  | 外旋 | 小さく前へならえをした位置から外側に回す動作 |
|  | 水平屈曲（水平内転） | 肩関節90°外転位から上腕を水平面で前方へ移動させる動き |
|  | 水平伸展（水平外転） | 肩関節90°外転位から上腕を水平面で後方へ移動させる動き |
| **脊柱** | 回旋 | 首をひねる動き（頸椎）、体をひねる動き（胸椎、腰椎） |

なお、**前腕の回外**（掌を上に向ける動作）および**前腕の回内**（掌を下に向ける動作）は、回旋の動作ですが、提供された表では「動きの面」は特定されていません。しかし、回旋運動は水平面の定義に含まれます。

このように、上半身の関節運動は、**前後方向の動き（屈曲/伸展）は矢状面**に、**左右方向の動き（外転/内転/挙上/下制/側屈）は前額面**に、そして**ひねりの動き（回旋）は水平面**に分類され、解剖学的に整理されています。
