## なぜこれが必要か(目的・背景)

体の不調や怪我の多くは、関節が本来持つべき役割（可動性・安定性）を果たせていないことに起因する。
本ガイドは、**ジョイントバイジョイント理論**に基づき、各関節の役割を正しく理解し、コンディショニングとトレーニングの質を高めることで、怪我を防ぎ、パフォーマンスを最大化することを目的とする。

## 基本(ジョイントバイジョイント理論の原則)

### 1. 関節の役割分担

関節は交互に「可動性（モビリティ）」と「安定性（スタビリティ）」の役割を持つ。

| 関節 | 役割 | 注意点 |
| --- | --- | --- |
| **胸椎** | **可動性 (Mobility)** | 回旋・伸展が必要。硬くなると腰椎に負担がかかる。 |
| **腰椎** | **安定性 (Stability)** | 過剰な動き（特に回旋・伸展）は腰痛の原因。 |
| **股関節** | **可動性 (Mobility)** | 球関節。力を貯める・発揮するために大きな可動域が必要。 |
| **膝関節** | **安定性 (Stability)** | 一方向にしか動かない関節。動きすぎるとストレスを受ける。 |
| **足首** | **可動性 (Mobility)** |  |
| **肩関節** | **可動性 (Mobility)** |  |
| **肩甲胸郭** | **安定性 (Stability)** | 肩関節の動きを支える土台として安定が必要。 |

### 2. 理論からの逸脱パターン

* **モビリティ・モビリティの連鎖**: 本来安定すべき関節（肩甲胸郭、膝、腰椎など）まで過剰に動いてしまう状態。
* *例*: ビハインドネックプレス（肩甲骨が動きすぎる）、膝が前に出すぎるスクワット。


* **代償動作**: 可動性が必要な関節（股関節、胸椎など）が動かないため、安定性が必要な関節（腰椎など）が無理に動いてカバーする状態。

## 実践：コンディショニングの順序とテクニック

### 理想的な実施フロー

1. **呼吸・リセット**: 動きを阻害する筋肉（大腿筋膜張筋など）を緩め、呼吸で安定性の土台を作る。
2. **安定性 (スタビリティ)**: 止めるべき場所（腰椎、膝など）を安定させる。
3. **可動性 (モビリティ)**: 動くべき場所（胸椎、股関節など）を動かす。
4. **運動連鎖**: 全身を連動させた動作へ移行する。

### 具体的なテクニック

* **トランクローテーション時の股関節屈曲**:
* **Why**: 足を伸ばすと腰椎まで回ってしまう。股関節を曲げることで腰椎をロック（安定化）し、胸椎のみを回旋させる。


* **四つ這い（キャットドッグ）**:
* **Why**: 股関節屈曲位で腰椎の過剰な動きを抑制し、胸椎の可動性を引き出す。

* **「どこを止め、どこを動かすか」**: 漫然と動かさず、常にこの意図を持つ。
* **鏡を見すぎない**: 視覚情報（外在的）に頼らず、身体感覚（内在的）に集中することで、関節の微妙なズレや動きを感じ取る。

* **膝先行スクワット**: 股関節の可動性を使わず、膝の安定性を犠牲にするためNG。

トレーニングは「筋肉を動かす」だけでなく、**「関節の役割を守る」**作業である。

本資料は、「ジョイントバイジョイント理論に基づくコンディショニングとトレーニング実践」と題された動画コンテンツの前半部分をレビューし、ジョイントバイジョイント理論の基礎、およびコンディショニングとトレーニングにおけるその応用と重要性についてまとめたものです。ジョイントバイジョイント理論は、体の各関節が持つべき機能（可動性または安定性）に基づいており、この理論を理解することで、運動中の体の動きをより効果的に制御し、怪我のリスクを軽減することができます。

### 主なテーマと重要事項

* ジョイントバイジョイント理論の基礎:  
* ジョイントバイジョイント理論は、体全体の連動性を理解し、各関節が持つべき固有の機能（可動性または安定性）に基づいてコンディショニングとトレーニングを行うことを提唱しています。  
* この理論によれば、特定の関節が適切に機能しない場合、本来安定しているべき関節が代償的に動きすぎてしまい、不調や怪我の原因となります。  
* 動画では、「胸を動かしたいです腰は安定していたい」「腰の上で胸が綺麗いに動きたいのにこの位置から全部が回線を行っている」という例を通して、理論からの逸脱が体の連動性を乱すことを説明しています (2:16-2:25)。  
* コンディショニングにおけるジョイントバイジョイント理論の応用:  
* コンディショニングでは、ジョイントバイジョイント理論に従い、特定の関節を安定させながら他の関節の可動性を引き出すことが重要です。  
* 股関節の屈曲位での体幹回旋: コンディショニングにおいて、股関節を伸展した状態で行う回旋運動は少ないことが指摘されています (0:42-0:49)。これは、体幹部の代償を防ぐために股関節を屈曲位にすることで重心の位置を安定させ、腰椎の過剰な動きを抑制するためです (0:50-1:21)。ワールドグレイテストストレッチや四つ這いのキャットドッグも、股関節の屈曲位を維持して行われる例として挙げられています (1:21-1:42)。  
* トランクローテーション: 体幹の回旋を目的とするトランクローテーションにおいても、足を伸ばした伸展位で行うと体全体が重心移動する動きになってしまい、体幹部と下半身のねじれが効果的に生まれません (3:53-4:06)。片足を屈曲させることで腰椎の安定性を保ち、胸椎の動きを促すことが強調されています (4:14-4:25)。  
* リリースとモビリティ: 動きを阻害している筋肉や筋膜のリリースは、トレーニングプロトコルの最初のフェーズとして重要です (8:50-9:03)。特に股関節の内旋を引き起こしやすい大腿筋膜張筋などが硬い場合、これを緩めずにコンディショニングを行っても効果が薄いことが述べられています (7:37-8:39)。これは、関節の適切な可動性が失われていると、本来安定すべき部位が代償的に動きすぎてしまうためです。  
* トレーニングにおけるジョイントバイジョイント理論の応用:  
* トレーニングにおいても、各関節の適切な機能を理解することが怪我の予防とパフォーマンス向上につながります。  
* ショルダープレスとビハインドネック: ショルダープレスやビハインドネックでのプルダウンなど、肩関節の過剰な可動性を要求する動きは、ジョイントバイジョイント理論から逸脱していると指摘されています (9:28-11:20)。肩関節は可動性に富む関節ですが、肩甲胸郭関節は安定性が必要な関節です (10:03-10:33)。ビハインドネックなどの動作では、肩甲胸郭関節が必要以上に動きすぎてしまい、「モビリティモビリティ」の繋がりとなり、肩の痛みを引き起こしやすくなります (10:33-11:08)。対照的に、通常のショルダープレスでは肩甲骨が安定した状態で肩の動きだけが出やすく、安全性が高いとされています (11:20-12:05)。  
* スクワット: スクワットにおける膝の動きについても言及されています (15:13-15:23)。膝が先行して動くスクワットは、膝関節に過剰なストレスをかける可能性があります (15:31-15:47)。これは、膝関節が本来安定性が必要な関節であるにも関わらず、動きすぎているためです (15:50-16:03)。スクワットでは、可動性の高い股関節を引き込む動きを意識することが重要であり、これにより股関節に力を蓄え、発揮することができます (16:06-16:32)。また、ワイドスクワットにおける膝の内向きの動きは、股関節の外旋に必要な力が不足していることを示唆しており、これにより膝周辺にストレスがかかることが説明されています (17:07-17:21)。  
* ヒップヒンジとデッドリフト: スクワットで上半身が倒れすぎる動きは、ヒップヒンジの動きが強すぎた結果であり、これはデッドリフトに近い動作となります (17:50-18:13)。これは、股関節の引き込みは強いものの、可動性が出すぎている状態であり、腰椎に負担がかかりやすいとされています (18:13-18:19)。  
* 適切なモビリティとスタビリティの重要性:  
* 適切なモビリティ（可動性）とスタビリティ（安定性）を獲得できている状態が、体の不調を防ぐ上で最も重要です (21:05-21:12)。  
* 可動性が出すぎた状態（ハイパーモビリティ）は、適切ではないと指摘されており、関節の可動域には適切な位置があることを理解する必要があります (21:28-21:48)。  
* 不調に気づくためにはジョイントバイジョイント理論を知っていることが不可欠であり、トレーニング中に応急処置的な修正を行うことも可能となります (22:36-23:06)。  
* 自己への意識とトレーニング:  
* 効果的なトレーニングのためには、鏡に頼りすぎず、自分の体内の感覚に意識を向けること（内的感覚）が重要です (26:18-26:37)。どの筋肉が動いているのか、どの関節が動いているのかという感覚を養うことが、パフォーマンス向上につながります (26:32-26:41)。

### まとめ

ジョイントバイジョイント理論は、コンディショニングとトレーニングにおいて、体の各関節の機能（可動性と安定性）を理解し、適切な動きを引き出すための重要な枠組みを提供します。股関節の屈曲位での体幹回旋や、股関節を意識したスクワットなど、具体的な例を通して、この理論に基づいた実践方法が示されています。また、ビハインドネックのような過剰な可動性を要求する動作の危険性や、動きを阻害する筋肉のリリース、そして自己の内的な感覚への意識の重要性も強調されています。この理論を理解し、自身の体と向き合うことで、より安全かつ効果的な運動が可能となることが示唆されています。ジョイントバイジョイント理論とは何ですか？  
ジョイントバイジョイント理論は、体の各関節にはそれぞれ「安定性（スタビリティ）」または「可動性（モビリティ）」のどちらか一方の役割が主にあるという考え方に基づいた理論です。この理論では、適切な体の機能のためには、それぞれの関節がその役割を適切に果たすことが重要であるとされています。例えば、足首は可動性が高く、その上の膝は安定性、その上の股関節は可動性が求められます。この連続性が体の効率的な動きを可能にします。

### なぜコンディショニングにおいてジョイントバイジョイント理論が重要なのでしょうか？

ジョイントバイジョイント理論を理解することで、コンディショニングの目的と方法をより深く理解できます。例えば、体幹の回旋を行うトランクローテーションという動きでは、股関節を屈曲位に保つことで、腰椎の過剰な動き（代償）を防ぎ、胸椎の可動性を引き出すことができます。これは、股関節の適切な屈曲が腰椎の安定性を高めるためです。このように、どの関節を安定させ、どの関節を動かすべきかを理解することで、より効果的で安全なコンディショニングが可能になります。漫然と体を動かすのではなく、「どこを止める、どこを動かす」という意図を持つことが重要です。

### トランクローテーションを行う際に股関節を屈曲させるのはなぜですか？

トランクローテーションなどの体幹回旋運動において股関節を屈曲位で行うのは、腰部の代償を防ぐためです。股関節を屈曲させることで骨盤周辺の密度が高まり、重心が安定します。これにより、体幹部が前方に移動しやすくなるのを抑制し、腰椎の過剰な回旋を防ぎます。その結果、胸椎の可動性をより効果的に引き出し、体幹と下半身のねじれを目的とした運動が可能になります。股関節を進展させた状態で行うと、回旋を止めるところがなくなり、全身が重心移動の動きになってしまい、腰椎に負担がかかるリスクが高まります。

### 四つ這いでのコンディショニングにおけるジョイントバイジョイント理論の応用例はありますか？

四つ這いでのコンディショニング、例えばキャットドッグのような動きにおいても、ジョイントバイジョイント理論は適用されます。この姿勢では、股関節は屈曲位を維持し、腰椎が過剰に動きすぎないようにコントロールされています。これにより、腰椎の安定性を保ちつつ、胸椎などの可動性を引き出すことができます。四つ這いの姿勢での回旋運動も、適切に行うためにはどこを安定させ、どこを動かすかという意識が重要になります。例えば、片側の股関節を制限することで、より効果的に体幹の回旋を促すことも可能です。

### トレーニングにおいてジョイントバイジョイント理論を無視するとどのような問題が起こりえますか？

トレーニングにおいてジョイントバイジョイント理論を無視すると、関節に過度な負担がかかり、怪我のリスクが高まります。例えば、ショルダープレスを頭の後ろで行うビハインドネックプレスや、ビハインドネックプルダウンは、肩関節や肩甲胸郭関節の可動性が過剰に出てしまいやすく、関節の挟み込みや痛みに繋がる可能性があります。これは、本来可動性が求められる関節（肩関節）と、安定性が求められる関節（肩甲胸郭関節）の両方が過剰に動いてしまう「モビリティ・モビリティ」の連鎖が起こるためです。推奨される前方でのショルダープレスやプルダウンは、肩甲骨が安定した状態で肩の動きだけを引き出すため、ジョイントバイジョイント理論に沿った安全な動作と言えます。

### スクワットにおける膝の動きとジョイントバイジョイント理論の関係を教えてください。

スクワットにおいて、膝が過剰に先行して動く動作は、膝関節に大きなストレスをかける可能性があります。ジョイントバイジョイント理論では、股関節は可動性、膝関節は安定性が求められます。膝が先行して動くことは、本来安定性が求められる膝が過剰に動いてしまう状態であり、膝への負担が増加します。適切なスクワットでは、まず股関節から引き込むことで股関節に力を溜め、膝関節は股関節の動きに連動して安定性を保ちながら動くことが理想的です。股関節の適切な可動性がないと、膝に負担がかかりやすくなります。

### 関節の「適切な可動域」とは具体的にどのような状態を指しますか？

関節の「適切な可動域」とは、ジョイントバイジョイント理論に基づき、その関節に求められる役割に応じた可動範囲を指します。例えば、可動性が求められる関節（股関節や胸椎など）はスムーズに動く必要がありますが、安定性が求められる関節（膝関節や腰椎など）は過剰な動きを抑制する必要があります。特定の関節の可動性が過剰すぎる「ハイパーモビリティ」の状態も、関節の逸脱や不調に繋がる可能性があります。例えば、女性によく見られる股関節の内旋（内股）や、過度な柔軟性による親指が腕につくような状態は、ハイパーモビリティの一例であり、必ずしも良い状態とは言えません。各関節にはその役割に応じた適切な可動域が存在します。

### なぜトレーニング中に鏡を見すぎない方が良いと言われるのですか？

トレーニング中に鏡を見すぎない方が良いと言われるのは、自己の内的な感覚に集中するためです。筋力トレーニングにおいて最も重要なことの一つは、「どの筋肉を動かしているか、どの関節を動かしているか」という自己の内的な感覚（プロプリオセプション）です。鏡を見ることは、外的な視覚情報に意識が向き、「フォームが崩れていないか」といった外面的な確認に終始しやすくなります。しかし、本当に重要なのは自分自身の体の中での感覚であり、その感覚を研ぎ澄ますことで、より効果的に特定の筋肉を活動させたり、関節の動きをコントロールしたりすることができます。パワーリフティング選手が鏡を使わずにトレーニングを行うのも、自己の内面に集中するためです。

ジョイントバイジョイント理論の基本概念  
ジョイントバイジョイント理論は、体の各関節が必要とする主要な機能（可動性または安定性）に基づいて、適切なコンディショニングとトレーニングを行うための考え方です。この理論では、特定の関節は可動性（モビリティ）が高くあるべきであり、別の関節は安定性（スタビリティ）が高くあるべきだと考えます。これらの機能が失われると、体の他の部分で代償的な動きが生じ、パフォーマンスの低下や怪我のリスク増加につながります。

* 可動性が必要な関節の例: 足首、股関節、胸椎、肩関節  
* 安定性が必要な関節の例: 膝関節、腰椎、肩甲胸郭関節

コンディショニングとジョイントバイジョイント理論  
適切なコンディショニングは、ジョイントバイジョイント理論に基づいています。これは、それぞれの関節が必要とする機能を回復または維持することを目的とします。

* 股関節の屈曲位での上半身回旋: 体幹部の代償を防ぐために行われます。股関節を屈曲させることで、重心が安定し、腰椎の過剰な動きを抑えながら胸椎の可動性を引き出します (1:00)。  
* 四つ這いでのキャットドッグ: 股関節を屈曲位に保ち、腰椎が過剰に動かないようにコントロールする運動です (1:24)。  
* ワールドグレイテストストレッチ: 後ろ足の股関節を伸展させた状態で行うと、腰部が解放的になりすぎて胸椎だけでなく腰椎まで回旋してしまうため、ジョイントバイジョイント理論から逸脱します (1:57)。適切に行うには、股関節を屈曲させた状態で行います (2:29)。  
* トランクローテーション: 体幹部と下半身の捻じれを作るモビリティエクササイズです。足を伸展させた状態で行うと、全体が重心移動の動きになってしまい、捻じれの効果が薄れます。片足を屈曲させることで腰椎のスタビリティを保ち、胸郭のモビリティを引き出します (3:55, 4:18)。  
* 筋肉の公式短縮（硬結）へのアプローチ: 代償を助長する筋肉の硬結を緩めることもコンディショニングの重要な要素です。特に股関節の内旋を伴う筋肉（大腿筋膜張筋など）が硬いと、股関節の内旋と骨盤の前傾が生じ、腰椎が前方に解放的になりやすくなります (7:44, 8:00)。立位での内旋制限がある場合、クラムシェルなどの運動だけでは不十分であり、原因となっている筋肉のリリースが必要です (8:19, 8:32)。リリースはトレーニングプロトコルの最初のフェーズに含まれるべきです (9:01)。

トレーニングとジョイントバイジョイント理論  
トレーニングにおいても、ジョイントバイジョイント理論に基づいた動作を選択することが重要です。

* ビハインドネックのショルダープレス/プルダウン: 肩関節と肩甲胸郭関節の両方で過剰な可動性（モビリティモビリティ）が生じやすく、肩関節前方へのインピンジメントや腰椎へのストレスを引き起こす可能性があります (10:37, 12:20, 13:00)。関節の適切な機能（肩関節はモビリティ、肩甲胸郭関節はスタビリティ）を維持できる範囲で行うことが推奨されます。推奨されるのは、肩甲骨が安定した状態で肩関節の動きを引き出す通常のショルダープレスやラットプルダウンです (11:24, 13:05)。  
* スクワット:膝先行のスクワット: 膝関節が必要以上に動きすぎている状態（モビリティ過多）であり、膝にストレスがかかりやすくなります (15:33)。  
* 股関節先行のスクワット（ヒップヒンジ）: 股関節の引き込みを意識することで、股関節に力を貯めることができます。これは股関節の適切な可動性を引き出す動きです (16:06)。  
* ワイドスタンススクワットでの膝の内側への倒れ込み: 股関節の外旋位で力が発揮できていない（お尻の力が入らない）状態であり、膝関節にストレスがかかります (17:07)。股関節の内転・内旋が優位になっている状態です (17:24)。  
* 上半身が倒れすぎるスクワット（グッドモーニングに近い動作）: 股関節の引き込みは強いものの、腰椎に負担がかかりやすく、腰椎の過剰な可動性が生じている可能性があります (18:04, 18:17)。  
* 適切な関節の機能: 適切な可動性と適切な安定性が確保されていることが重要です。ハイパーモビリティ（過剰な可動性）は不調の原因となります (21:01, 21:30)。

コンディショニング・トレーニングの実施において重要なこと

* 意図を持って運動する: どこを止め、どこを動かすのかという意識を持つことが重要です (5:03)。ただ漫然と行うのではなく、目的を理解して行うことで効果が変わります (5:10)。  
* 感覚を重視する: 鏡に頼りすぎず、自分の体内の感覚（どの筋肉が働いているか、どの関節が動いているか）に意識を向けることが重要です (26:25)。これは内在的な集中と呼ばれます (26:29)。  
* 原因の特定: 不調の原因がジョイントバイジョイントの崩れによるものか、日常の環境や感覚の問題など他の要因によるものかを考慮する必要があります (22:18)。ジョイントバイジョイント理論は不調に気づくための有効な手段であり、トレーニング中の応急処置的な改善にも役立ちます (22:36, 22:41)。

重要用語集

* ジョイントバイジョイント理論 (Joint-by-Joint Theory): 体の各関節が必要とする主要な機能（可動性または安定性）に基づいて、コンディショニングとトレーニングを設計する考え方。  
* コンディショニング (Conditioning): 運動のパフォーマンス向上や怪我予防のために、体の状態を整えること。筋力、柔軟性、バランス、可動性などを改善する。  
* トレーニング (Training): 特定の目標（筋力向上、持久力向上など）を達成するために、計画的に運動を行うこと。  
* モビリティ (Mobility): 関節が動くことのできる範囲と、その範囲をコントロールする能力。可動性。  
* スタビリティ (Stability): 関節や体幹が、外部からの力や自身の動きに対して安定している状態。安定性。  
* 代償 (Compensation): 体の一部の機能不全を、他の部分が過剰に働くことで補うこと。多くの場合、不適切な運動パターンや怪我につながる。  
* 股関節屈曲 (Hip Flexion): 股関節を曲げる動き。  
* 股関節伸展 (Hip Extension): 股関節を伸ばす動き。  
* 上半身回旋 (Trunk Rotation): 体幹部を捻じる動き。トランクローテーションともいう。  
* 腰椎 (Lumbar Spine): 背骨の腰の部分。  
* 胸椎 (Thoracic Spine): 背骨の胸の部分。  
* 肩甲胸郭関節 (Scapulothoracic Joint): 肩甲骨と胸郭（肋骨）の間にある機能的な関節。肩甲骨の動きに関連する。  
* ワールドグレイテストストレッチ (World's Greatest Stretch): 体の複数の部位を同時にストレッチする複合的な運動。  
* 大腿筋膜張筋 (Tensor Fasciae Latae): 股関節の外側にある筋肉。股関節の屈曲、外転、内旋に関わる。  
* 骨盤前傾 (Anterior Pelvic Tilt): 骨盤が前方に傾くこと。  
* リリース (Release): 硬くなった筋肉や筋膜を緩めること。セルフケアや治療で行われる。  
* ショルダープレス (Shoulder Press): 肩関節を屈曲・外転させ、腕を頭上方向に挙上する運動。  
* プルダウン (Pulldown): バーなどを上から引き下げる運動。主に背中の筋肉を鍛える。  
* ビハインドネック (Behind the Neck): 首の後ろで動作を行うこと。ショルダープレスやプルダウンなどで行われる形式。  
* インピンジメント (Impingement): 関節内で組織が挟み込まれて痛みが生じる状態。  
* スクワット (Squat): 膝と股関節を曲げてしゃがみ込み、再び立ち上がる運動。  
* ヒップヒンジ (Hip Hinge): 股関節を支点として、体幹部を前傾させる動き。デッドリフトやグッドモーニングなどの運動で重要。  
* ワイドスタンス (Wide Stance): スクワットなどで足を広く開いたスタンス。  
* ハイパーモビリティ (Hypermobility): 関節の可動域が正常範囲を超えて過剰に大きい状態。  
* 運動連鎖 (Kinetic Chain): 体の一部の動きが、他の部分の動きに連鎖的に影響を与えること。上行性（下から上）と下行性（上から下）の運動連鎖がある。  
* 内在的な集中 (Internal Focus): 自分の体内の感覚（筋肉の収縮、関節の動きなど）に意識を向けること。  
* 外在的な集中 (External Focus): 体の外にある目標や結果に意識を向けること（例: バーを持ち上げること、特定の高さまですることなど）。

「可動性（モビリティ）と可動性（モビリティ）の繋がり」が悪いとされるのは、**ジョイント・バイ・ジョイント理論の基本的な原則から逸脱し、関節や周囲の組織に過度なストレスを集中させる**ためです \[10:33, 10:37, 11:11, 21:05, 21:10\]。

筋力トレーニングやコンディショニングは、この理論に基づいて「どこを止める、どこを動かす」という意図を持って行う必要がありますが、モビリティ・モビリティの繋がりは、**体が過剰に動きすぎている状態**を指します \[7:00, 10:39, 11:14, 11:16\]。

### **1\. ジョイント・バイ・ジョイント理論からの逸脱**

ジョイント・バイ・ジョイント理論では、身体の関節は、\*\*安定性（スタビリティ）\*\*が求められる関節と、**可動性（モビリティ）が求められる関節が交互に連鎖**している状態が理想とされています \[21:05, 21:10\]。

* **理想的な状態:** 例えば、可動性が求められる胸椎（胸郭）と、安定性が求められる腰椎（腰部）が隣り合っている、といった状態です \[2:16, 2:18\]。  
* **モビリティ・モビリティの問題:** モビリティが求められる関節に、隣接する関節もモビリティを出してしまい、「**モビリティ・モビリティ**」という動きすぎの状態の連鎖が起こると、本来必要とされる**安定性の要素が失われます** \[10:37, 11:11, 21:01, 21:05\]。

### **2\. 怪我や炎症のリスクの増大**

安定性が失われた状態で動作を繰り返すと、関節の構造が適切に機能せず、骨が逸脱した動きになり、怪我や痛みのリスクが高まります \[3:16, 12:50, 12:51\]。

#### **例：肩関節周辺での問題**

ビハインドネック動作（頭の後ろにバーを持ってくる動作など）は、このモビリティ・モビリティの繋がりが問題となる典型的な例です \[10:00, 10:42\]。

1. **肩関節（モビリティ）:** 肩関節自体は**モビリティ**動作が広い関節です \[10:03, 10:06\]。  
2. **肩甲胸郭関節（本来はスタビリティ）:** 肩甲骨の裏側、肋骨に乗っている**肩甲胸郭関節**は、本来**安定性**が必要な部位です \[10:25, 10:28, 10:33\]。  
3. **連鎖の発生:** ビハインドネック動作を行うと、肩関節のモビリティ運動に加え、本来安定性が必要な**肩甲胸郭関節まで動きすぎてしまい**、「モビリティ・モビリティ」の繋がりが発生します \[10:18, 10:31, 10:35, 11:11, 11:14\]。  
4. **結果:**  
   * **挟み込み（インピンジメント）:** 可動域が取れない人がこの動作を行うと、肩の筋肉が**前方に移動**しやすくなり、憲法（けんぽう）という部分が挟まりやすくなるため、**痛みが出やすい**です \[10:44, 10:50, 10:52, 10:55\]。  
   * **炎症:** 動きが逸脱した状態で繰り返すと、体に対して骨が擦れてきて**炎症**につながる可能性があります \[11:02, 11:05, 11:08\]。

### **3\. パフォーマンスの低下と代償動作**

過剰な可動性（ハイパーモビリティ）が出ると、元々持っている安定性の自由度が逸脱するため \[20:51, 21:01, 21:05\]、体の制御が難しくなり、意図しない**代償動作**（動きの逃げ）が生じます。

このため、**適切なモビリティと適切なスタビリティ**を関節ごとに獲得できていなければ、不調が起こる可能性が高くなります \[21:07, 21:12\]。

---

モビリティ・モビリティの繋がりが悪いというのは、例えるなら、**電車の車両と車両を繋ぐ連結器が両方とも柔軟すぎる状態**に似ています。

連結器（関節）は片方が固定され（安定性）、もう片方が柔軟に動く（可動性）ことで、全体が安定した状態でカーブ（動作）を曲がることができます。しかし、両方の連結器が自由に動きすぎてしまうと、走行中に車両同士が激しくぶつかり合ったり（インピンジメント）、脱線（逸脱）するリスクが高まってしまいます。

ご提示いただいたソースに基づくと、**股関節を屈曲位（曲げた状態）で運動を行うこと**（特に体幹回旋などのコンディショニング）が防ぐ主な代償は、\*\*腰椎（腰部）の過剰で解放的な動き（代償）\*\*です \[0:53, 1:00, 2:05\]。

#### **1\. 呼吸とリセット・リリース（準備フェーズ）**

* **呼吸の重要性:** 安定性（スタビリティ）は呼吸と深く関わっています \[24:26\]。しっかりと息を吐けている状態は、良いスタビリティ（安定性）につながります（吐きすぎるとハイパーモビリティになってしまう場合もあります） \[24:26, 24:32\]。  
* **リセットとリリース:** 呼吸の次に、\*\*特定の筋肉（例：大腿筋膜張筋など、内旋や骨盤前傾を伴い腰椎を不安定にする筋肉）\*\*の拘縮・短縮を緩めてあげることが必要です \[7:31, 7:45, 8:10, 8:34, 8:36\]。  
* **目的:** 呼吸のエクササイズとリリースのエクササイズは、**次のムーブメントのエクササイズやコンディショニングに入りやすくするため**に、固くなりやすいところを緩めてあげるために必ず行います \[9:06, 9:16\]。

#### **2\. 安定性（スタビリティ）の確保（体幹部を中心として）**

* **優先順位:** 安定性から入る理由の一つは、呼吸が安定性に関わるためです \[24:26\]。また、地面に接している面（例：足部、足首）やその上部にある部位（例：腰椎）など、身体を支える部分の安定性を確認・強化します \[25:12, 25:22\]。  
* **「どこを止めるか」の意識:** ジョイントバイジョイント理論を理解し、\*\*「どこを止める」\*\*かを意識しながら動作を行うことが、コンディショニングの質を決定的に変えます \[6:59, 7:05\]。

#### **3\. 可動性（モビリティ）の獲得と調整**

安定性が確保されたら、次に\*\*可動性（モビリティ）\*\*を出すことに焦点を当てます \[24:40\]。

* **「どこを動かすか」の意識:** 胸椎や股関節など、本来可動性が求められる関節が、安定性を保った部位の上で**解放的に動ける状態**（例：腰椎のスタビリティを保った状態での胸郭の回旋）を目指します \[2:16, 2:39, 4:22\]。

#### **4\. 運動連鎖（ムーブメントの統合）**

安定性と可動性が個々の関節で適切に整った後、**運動連鎖**（上行性および下降性の運動連鎖を含む）へと進みます \[24:45, 28:06, 28:17\]。

* **相乗効果:** どこか一つの関節の安定性または可動性が良くなれば、その上の関節にも下の関節にも**良い影響を及ぼす**ため、統合的な動作の中で、各関節が適切な役割を果たせるように訓練します \[20:37, 20:42, 20:51\]。

### **まとめ**

意図を持った動作は、コンディショニングを単なる運動ルーティンから、\*\*「ジョイントバイジョイント理論に基づき、どこを安定させ、どこを稼働させるかを意識し、その感覚を自分の中に探求する」\*\*活動へと変貌させます。これにより、不調に気づき、適切な修正が可能となり、コンディショニングの効果を決定的に高めるのです。

---

この「意図を持った動作」の重要性は、まるで**カーナビに目的地を設定すること**に似ています。

目的地（どこを安定させ、どこを動かすかという意識）を設定せず、ただ車（体）を走らせる（動かす）だけでは、遠回りしたり、思わぬ場所で事故（怪我や代償）を起こすリスクがありますが、目的地を明確に設定し、理論（ジョイントバイジョイント）に従って集中して運転（動作）することで、安全に、かつ最短距離で質の高い結果に到達できるからです。

ジョイント・バイ・ジョイント理論（Joint-by-Joint Theory）が定める安定性（スタビリティ）と可動性（モビリティ）の役割は、**身体の各関節が特定の機能を持つ**という原則に基づいて理解することができます。この理論を理解することで、コンディショニングやトレーニングにおいて「**どこを止める**（安定させる）」「**どこを動かす**（可動させる）」べきかが明確になります。

以下に、ソースに基づき、主要な関節における安定性と可動性の役割とその関連性についてご説明します。

### **1\. 安定性と可動性の基本的な役割分担**

ジョイント・バイ・ジョイント理論では、関節は安定性が求められる部位と、可動性が求められる部位が交互に連なっていると見なされます。

* **可動性（モビリティ）の役割：** 動作の自由度が高く、積極的に動かすべき関節です。  
* **安定性（スタビリティ）の役割：** 動作中に過剰に動きすぎないように、しっかりと固定し、安定性を保つべき関節です。

この理論における理想的な状態は、**適切なモビリティと適切なスタビリティ**が獲得されていることです。

### **2\. 具体的な関節における役割**

ソースでは、体幹部や下肢、肩周りの動きについて具体的な役割分担が説明されています。

| 関節部位 | 求められる主な役割 | 役割が逸脱した場合（例） | 参照 |
| ----- | ----- | ----- | ----- |
| **胸椎 (胸)** | 可動性（モビリティ） |  |  |
| **腰椎 (腰)** | 安定性（スタビリティ） | 過剰に動きすぎると腰が悪くなる。胸を回線したときに腰まで開いてしまう。 |  |
| **股関節** | 可動性（モビリティ） | 股関節を引き込めないと、力を貯めたり、組み立てたりできない。 |  |
| **膝関節** | 安定性（スタビリティ） | 動きすぎると（膝が先行して動くと）ストレスを受けやすい。 |  |
| **肩関節** | 可動性（モビリティ） |  |  |
| **肩甲胸郭関節** | 安定性（スタビリティ） |  |  |

**【体幹部の理解】** トランクローテーション（体幹回旋）の動作においては、**腰椎のスタビリティを保った状態**で**胸（胸椎）が動く**ことが必要とされます。腰の上で胸が綺麗に動くことが理想です。もし腰椎が安定しない状態で上半身を回線しようとすると、腰まで開いてしまい、胸を動かしたいのに腰は安定していたいというジョイント・バイ・ジョイントの理論から逸脱した動きになります。

**【下肢の理解】** スクワットの例では、股関節は球体であり**可動性を出す**必要があり、力を貯める中心的な場所です。一方、膝関節は**安定性で固めた方が良い**部位です。膝が先行して動きすぎると（前方に）、膝にストレスがかかりやすくなります。 **股関節が良い動きを出せれば**（適切なモビリティ）、**腰部には良い安定性を供給**でき、**膝にも良い安定性を供給**できるとされています。

**【肩周りの理解】** 肩関節はモビリティ動作が広い関節ですが、肩甲胸郭関節（肋骨に乗っている肩甲骨の裏側）は安定性が必要とされます。通常のショルダープレスは、**肩甲骨が安定した状態**で**肩の動きだけ**が出ているため推奨されます。

### **3\. 安定性と可動性の逸脱（崩壊）がもたらす問題**

ジョイント・バイ・ジョイント理論が崩壊する原因の一つは、**モビリティ・モビリティ**のつながりです。

1. **可動性が過剰になること（ハイパーモビリティ）：** 安定性が必要な関節（腰椎や膝関節など）が動きすぎたり、可動性が求められる関節（股関節や胸椎など）が過剰に動きすぎたりする（ハイパーモビリティ）と、身体に不調や痛みが生じるリスクが高まります。 例えば、ビハインドネックでのショルダープレスやプルダウンは、肩関節のモビリティと肩甲胸郭関節のモビリティが連なり過ぎてしまう（モビリティ・モビリティ）ため、肩の筋肉が前方に移動し、憲法が挟まるなどして痛みが出やすい動作とされています。これは、動きが逸脱すると、痛くなくても繰り返すことで炎症につながるためです。  
2. **安定性の欠如：** 全身が動きすぎると、結果的に腰部などが過剰に動きすぎて骨が一脱した動きになり、腰を痛めるリスクにつながります。  
3. **予防と改善：** この理論を知ることで、なぜ特定の動作で代償（他の部位で動きを補うこと）が発生しているのかを理解し、**動きを邪魔している筋肉や筋膜のリセット・リリース**を行うといった予防策を講じることができます。

ジョイント・バイ・ジョイント理論は、自身のトレーニングにおいて「どこを止める、どこを動かす」といった意識を持つことで、コンディショニングの質を大きく変えることができる、修正可能な理論であるとされています。

---

**理解を深めるための比喩:** ジョイント・バイ・ジョイント理論における安定性と可動性の役割は、まるで\*\*電車（列車）\*\*の連結器と車輪の関係に似ています。

可動性（モビリティ）が求められる関節は**車輪**であり、スムーズに回転し、進行方向を変える柔軟性が必要です。一方、安定性（スタビリティ）が求められる関節は**連結器**であり、車輪の動きを制御し、全体がバラバラにならないようにしっかりと固定する役割を担っています。もし、連結器（安定性が必要な部位）までが車輪のようにぐらぐらと動きすぎてしまう（モビリティ・モビリティ）と、列車全体が脱線したり、軋んだりして、効率的な移動（動作）ができなくなってしまうのです。
