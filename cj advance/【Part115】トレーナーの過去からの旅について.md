# パーソナルトレーナー業界の変遷と現在地

> クロニクルジャパン内部対談。2010年代から2026年現在までのトレーナー業界の構造的変化、独立の罠、組織化の必要性を解説した音声をmd化。経営判断・採用・独立支援に活用。

## 1. トレーナーのキャリアパス変遷（2010年代前半）

### 当時の標準ルート

```
高卒 or 大卒
  → 専門学校（2〜3年）
  → NSCA / JATI などの民間資格取得
  → フィットネスクラブ就職 or チームスポーツのAT
  → 月給 20〜25万円
```

### 10年前の資格事情
- 当時は「トリプル資格（NSCA + 国家資格 + AT）」を持てば最強と言われた
- 資格ブームの背景は**学校・資格団体の会員増加が目的**
- 今はNSCAが最短3日で取得可能→資格の価値は相対的に低下

## 2. マイクロジムの誕生（2013年〜）

### 独立の流れ
- フィットネスクラブで月150本（1回5,000〜6,000円）をこなすと「自分は最強」感覚になる
- 仲の良いお客様10〜20人がついた段階で独立衝動が発生
- パトロン（資金提供者）が現れ、100〜200万円規模のマイクロジムが急増

### マイクロジムの構造的問題
- 1部屋・1人経営の上限は**月70〜90万円**（1回6,000円×160本）
- スタッフを雇うと給与20〜30万円を渡した後の取り分が激減
- 雇った人材が成長するとそのまま独立→**永遠のリセットサイクル**

```
自分がトレーニング → お客様獲得 → スタッフ雇用
  → スタッフにお客様つく → スタッフ独立
  → また0から雇用 → 繰り返し
```

---

## 3. ピラティスブームと同じ構造の繰り返し（2020年代）

### ピラティス参入の特徴
- 資格費用：20〜100万円（学校不要・個人でも取得可能）
- Instagramで集客 → マイクロジムを開業
- 「トレーニング × ピラティス × 整体」の掛け算で差別化を図るが…

### 問題の本質は変わらない
- 結局「2人でジムを開いたら絶対揉める」
- 個人のキャパ（1部屋・1人）の上限は変わらない
- 現在ピラティスジムも潰れ始めている
- **撤退コストがかかる**（スケルトン戻しに200万円以上）

### 次のトレンド予測
- ヨガが来る可能性が高い（器具不要・コスト低・呼吸法の親和性）

## 4. 個人事業主の限界と組織化の必要性

### 個人経営の天井

| 状況 | 問題 |
|---|---|
| 病気・怪我 | 即売上ゼロ |
| 家族を持てない | 休めないため |
| SNSで伸びない限り | 月+20万が限界 |
| 地方で1,200万超え | ほぼ前例なし |

### 組織化のメリット（クロニクルジャパンの事例）
- 店舗を任せる → 2〜3人チームで150〜200万売上
- 売上に応じて給与を上乗せ（50〜60万渡せる体制）
- 1人が抜けても他でカバーできる
- 福岡・大阪・博多・黒崎・熊本に展開可能
- **「独立したい」人には経験を積ませ、準備が整ったら任せる**

### ビジョン（福岡のジム事例）
- 法人契約で大量集客（1社10名以上）
- パーソナル料金：3,300〜6,600円/回
- 1トレーナーが150万売り上げ → 手取り約60万
- ただし「自分が倒れたら終わり」構造は変わらない

## 5. SNS・メディアと収益構造の変化

### プラットフォームの変遷

| 時代 | ツール | 特徴 |
|---|---|---|
| 〜2018年 | Instagram黎明期 | 不特定多数への情報発信が強かった |
| 2018年〜 | 情報商材ブーム | 1,000フォロワー×1%購入×30万円＝300万円 |
| 現在 | YouTube・Note・TikTok | 動画 + 長文コンテンツの組み合わせ |

### 情報商材の終焉
- 「フォロワー多い＝稼げる」時代は終わりつつある
- トップ層でないと月+20万が限界

---

## 6. AI時代の見通し

- AI が進化しても「触る技術」「身体へのアプローチ」は代替困難
- フィットネス・整体・リハビリ系は**残る仕事として有力**
- ただし**組織として動けるかどうか**が生き残りの分岐点

---

## 7. 現実的な収益シナリオ（個人〜チーム）

| フェーズ | 月収目安 | 条件 |
|---|---|---|
| 個人（1部屋・満員） | 70〜90万円 | 週5・1日8本・単価6,000円 |
| 個人＋副収入 | +20万円 | ノート・物販・コンテンツなど |
| チーム（2〜3人） | 150〜200万売上 | 組織化・エリア展開 |
| 年収1,000万超え | 要SNS影響力 or 資本 | 関東圏 or 複数拠点 |

---

## 8. 独立を考えているトレーナーへの示唆

- **小屋（マイクロジム）を出しても構造は変わらない**
- 20代なら体力で乗り越えられるが30代以降は限界が来る
- 地方で小さくやる方が持続可能性が高い
- 組織に入りながら**ノウハウを学んでから独立**する選択肢が合理的
- 「お金を稼ぐ」ことに罪悪感を持たないことが重要
- 物販は収益の天井が高くなる可能性がある
